# Campus Opportunities Hub

This is a simplified platform designed for students and colleges to find and share Hackathons, Internships, and Events. The goal is to cut out the noise of big platforms like LinkedIn or Internshala and keep it focused on student-friendly opportunities.

__The Plan__

I'm building this in 30 days. The focus is on a clean UI (think BookMyShow cards) and a fast user experience. No clutter, just the data students actually need.

__Tech Stack__

I'm starting with the basics and moving into a full-stack modern environment:

Frontend: HTML/CSS/JS (Initially), moving to React and Tailwind CSS.
Backend: Node.js & Express.
Database: MongoDB.
